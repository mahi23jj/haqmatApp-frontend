# Haqmate

Haqmate is a teff selling app that lets users buy different teff qualities
directly from the company.

## Features

- Product search and browsing
- Order flow with cart and checkout
- Payments via Chapa and Telebirr (additionaly supports payment proof by screenshot)
- Authentication
- Reviews and ratings

## What is done well

- Authentication flow is complete and consistent
- Cart and order flow are clear and reliable
- Payments are integrated and user friendly
- State management is organized and predictable
- Search works smoothly across products

## Tech stack

- Flutter (Dart)
- Android, iOS, Web, Windows, Linux, macOS targets

## Getting started

1. Install Flutter SDK and set up your environment.
2. Fetch dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```

## Project structure

- lib/ - main application code
- assets/ - images and static assets
- android/, ios/, web/, windows/, linux/, macos/ - platform targets
